文件位置：`njusttt.cls`，函数 `\makefrontcover`（约第 168-220 行）

## 可调参数速查

### 一、作者/学号块（居中显示姓名和学号的区域）

```latex
\begin{minipage}{0.45\textwidth}          % ← [A] 整块宽度，越大越宽
  \centering
  {\kaibf\zihao{-2}                       %   字体：楷体加粗小二号
    \underline{\makebox[0.8\linewidth][c]{\NJUST@value@author}}}  % ← [B] 下划线宽度
  \par\vskip 2pt                          % ← [C] 下划线到"(学生姓名)"间距
  {\songti\zihao{5}（学生姓名）}           %   小字标注
  \par\vskip 16pt                         % ← [D] 姓名块和学号块之间的间距
  {\kaibf\zihao{-2}
    \underline{\makebox[0.8\linewidth][c]{\NJUST@value@studentid}}}
  \par\vskip 2pt
  {\songti\zihao{5}（学号）}
\end{minipage}
```

| 标记 | 当前值 | 作用 |
|------|--------|------|
| [A] | `0.45\textwidth` | 作者块整体宽度，调大=姓名框变宽 |
| [B] | `0.8\linewidth` | 下划线占块宽的比例，0.8=80% |
| [C] | `2pt` | 下划线到下方小字"(学生姓名)"的距离 |
| [D] | `16pt` | 姓名行和学号行之间的垂直间距 |

### 二、底部信息表格（指导教师、学院、专业等）

```latex
\begin{tabular}{@{} p{3.8cm} p{9cm} @{}}  % ← [E] 左列宽   ← [F] 右列宽

  % 指导教师行（特殊：名字下划线较短，职称在后面）
  \makebox[3.8cm][s]{指\hspace{0.3em}导\hspace{0.3em}教\hspace{0.3em}师} &   % ← [G] 字间距
    \underline{\makebox[3cm][c]{张三\quad教授}} \\[10pt]  % ← [H] 名字下划线宽  ← [I] 行距

  % 其他行（统一：9cm 全长下划线）
  \makebox[3.8cm][s]{学\hspace{0.3em}生\hspace{0.3em}学\hspace{0.3em}院} &
    \underline{\makebox[9cm][c]{...}} \\[10pt]

  \makebox[3.8cm][s]{专\hspace{0.3em}业} &
    \underline{\makebox[9cm][c]{...}} \\[10pt]

  \makebox[3.8cm][s]{研\hspace{0.3em}究\hspace{0.3em}方\hspace{0.3em}向} &
    \underline{\makebox[9cm][c]{...}} \\[10pt]

  \makebox[3.8cm][s]{提\hspace{0.3em}交\hspace{0.3em}时\hspace{0.3em}间} &
    \underline{\makebox[9cm][c]{...}}
\end{tabular}
```

| 标记 | 当前值 | 作用 |
|------|--------|------|
| [E] | `3.8cm` | 左列（标签）宽度，调大=标签字更散、右列变窄 |
| [F] | `9cm` | 右列（下划线）宽度，控制下划线长度 |
| [G] | `\hspace{0.3em}` | 标签每个字之间的间距，增大=字更散 |
| [H] | `3cm` | 指导教师名字下划线宽度（比较短，后面跟职称） |
| [I] | `10pt` | 表行之间的垂直间距，所有行统一 |

> **注意**：`\makebox[3.8cm][s]{...}` 中的 `[s]` 表示"分散对齐"——会把文字拉伸填满 3.8cm 宽度。如果标签只有两个字（如"专业"），拉伸效果会很明显；四个字（如"指导教师"）则较均匀。如果某个标签看起来太散或太挤，单独调整它内部的 `\hspace` 值。

### 三、各区块之间的垂直间距

```latex
%% Logo → "毕业设计（论文）报告"
\vskip 20pt                              % ← [J]

%% "毕业设计（论文）报告" → 论文标题
\vskip 36pt                              % ← [K]

%% 论文标题 → 作者/学号
\vskip 36pt                              % ← [L]

%% 作者/学号 → 底部表格
\vskip 28pt                              % ← [M]
```

| 标记 | 当前值 | 所在区块之间 |
|------|--------|-------------|
| [J] | `20pt` | Logo 下方到"毕业设计（论文）报告" |
| [K] | `36pt` | 报告类型到论文标题 |
| [L] | `36pt` | 论文标题到作者学号块 |
| [M] | `28pt` | 作者学号块到底部信息表格 |

### 四、Logo 大小

```latex
\includegraphics[width=0.7\textwidth]{fig/logo/njust.eps}
```
`0.7\textwidth` = 页面文字宽度的 70%。调大=Logo 变大。

## 调整思路

- **封面整体太挤** → 增大 [J][K][L][M] 的 `\vskip` 值
- **封面整体太空** → 减小 [J][K][L][M] 的 `\vskip` 值
- **作者姓名偏左/偏右** → 调 [A] minipage 宽度
- **下划线太长/太短** → 调 [B] 比例 或 [F][H] 的 cm 值
- **标签字太散/太挤** → 调 [G] `\hspace` 值
- **表格行太挤** → 增大 [I] `\\[10pt]`
- **某一行与其他行不对齐** → 检查该行的 `\makebox` 宽度参数
